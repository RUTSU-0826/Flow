# FlowCloneProject

## 프로젝트 소개
팀 협업 및 업무 관리를 위한 협업툴 ‘플로우(Flow)’를 클론한 프로젝트입니다.  
Java Servlet과 JSP를 활용한 순수 MVC 패턴 기반으로 구현했습니다.

## 주요 기능
- 업무 게시판 CRUD  
- 댓글 및 실시간 알림 기능  
- 프로젝트 및 일정 관리  
- 사용자 권한 및 팀 관리  

## 담당 역할 및 구현 기능
- 피드 게시판 구현  
- 업무 페이지 구현  
- 대시보드 구현  
- 실시간 채팅 기능 구현  
- 프로젝트 생성 및 수정 기능 구현  
- 로그인 및 회원가입 기능 구현  

## 사용 기술
- Java Servlet / JSP (MVC 패턴)  
- Oracle Database  
- WebSocket (실시간 알림 및 채팅)  
- Apache Tomcat
